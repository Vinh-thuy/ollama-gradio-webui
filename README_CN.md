# ollama-gradio-webui
本项目提使用`gradio`提供一个用户友好的聊天界面，用于与各种`Ollama`模型交互。
## 如何运行
* 安装 [Ollama](https://github.com/ollama/ollama)
* 创建 conda 环境 `conda create -n ollama python=3.12`
* 激活 conda 环境  `conda activate ollama`
* 安装依赖 `pip intall -r requirements.txt`
* 运行 `python app.py`
